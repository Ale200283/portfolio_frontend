Ankun

Creative software

Este proyecto nace con la iniciativa de crear software para dar soluciones a los problemas cotidianos de usuarios finales, proveyendo a estos de las herramientas necesarias para reflejar la información de forma fehaciente y concisa de todos los movimientos de su negocio.

Fecha inicial del proyecto: 21/06/2022

Objetivo:

  • Asesorar sobre la adquisición de hardware para el desarrollo de negocio
  • Brindar el software necesario con la finalidad de reflejar la información de forma trasparente.
  • Trabajar en conjunto con el usuario final, mejorando así los tiempos de respuestas.
